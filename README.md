# Install
- postgres
- ruby
- bundler
- docker

## Commands
- install bundler `gem install bundler`
- install gems: `bundle install`
- create database: `rails db:create`
- run migrations: `rails db:migrate`
- seed database with data: `rails db:seed`
- start server: `rails s`
